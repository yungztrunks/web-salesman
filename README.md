```txt

TYPE: TSP
COMMENT: 52 locations in Stuttgart Badcanstatt
DIMENSION: 52
EDGE_WEIGHT_TYPE: EUC_2D
NODE_COORD_SECTION
1 565.0 575.0
2 25.0 185.0
... # and so one
52 1.0 1.0
EOF
```
